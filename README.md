# Module 11 Assignment: Data Visualization with Matplotlib

This repository focuses on **Data Visualization** for SunCoast Retail. The project transforms complex datasets into actionable business insights through a variety of graphical representations.

## Files Included
* `module11-assignment.py`: The main Python script used to generate retail datasets and create a multi-plot business dashboard.

## What I Practiced
* **Trend Analysis:** Plotting quarterly sales growth and seasonal spikes (notably in Q4) over a two-year period.
* **Comparative Visuals:** Using grouped bar charts and stacked percentage bars to compare category performance across different city locations.
* **Relationship Mapping:** Creating scatter plots with regression lines to analyze the correlation between advertising spend and sales.
* **Demographic Profiling:** Utilizing histograms and box plots to visualize customer age distributions and purchasing habits.
* **Dashboard Design:** Consolidating multiple metrics into a single "Executive Dashboard" using Matplotlib’s subplots for professional reporting.
